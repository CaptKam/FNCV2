## DESIGN SYSTEM — "The Ethereal Archivist"

### Design Philosophy
This is NOT a flat app. It is spatial. Treat the UI as stacked sheets of fine paper and glass. Depth comes from TONAL LAYERING (background color shifts), not drop shadows. The "No-Line Rule" is absolute: never use 1px solid borders to separate sections. Structure is defined through surface color changes and generous whitespace.

### File: constants/typography.ts
Two font families loaded via expo-google-fonts:
- Noto Serif: editorial warmth, headlines, display text
- Inter: functional precision, body text, labels, buttons

Export named style objects:
```
displayLarge:    Noto Serif 700, 56px, letterSpacing: -2%   (hero statements)
displayMedium:   Noto Serif 700, 40px, letterSpacing: -1.5% (country names on heroes)
display:         Noto Serif 700, 28px                       (screen titles)
headlineLarge:   Noto Serif 700, 32px                       (section headers)
headline:        Noto Serif 700, 22px                       (card titles)
title:           Noto Serif 500, 18px                       (recipe names)
titleMedium:     Inter 600, 17px                            (button text)
titleSmall:      Inter 600, 14px                            (chip text)
body:            Inter 400, 17px, lineHeight: 1.6           (instructions)
bodySmall:       Inter 400, 14px                            (helper text)
caption:         Inter 500, 12px                            (timestamps)
labelLarge:      Inter 700, 10px, letterSpacing: +5%, uppercase (section labels like "TONIGHT'S PLAN")
labelSmall:      Inter 600, 10px, letterSpacing: +3%, uppercase (metadata labels)
```

The "Caption over Title" editorial pattern: pair a `labelLarge` uppercase line ABOVE a `headline` or `display` — this is the hallmark of the Ethereal Archivist look.

### File: constants/shadows.ts
Shadows are a LAST RESORT. Use tonal layering first. When you must float:
```
ambient:    { shadowOffset: {width:0, height:20}, shadowOpacity: 0.06, shadowRadius: 40, shadowColor: '#1D1B18' }
subtle:     { shadowOffset: {width:0, height:4},  shadowOpacity: 0.04, shadowRadius: 16, shadowColor: '#1D1B18' }
hero:       { shadowOffset: {width:0, height:8},  shadowOpacity: 0.20, shadowRadius: 24, shadowColor: '#9A4100' }
```
Shadow color is ALWAYS a tinted version of onSurface (#1D1B18), NEVER pure black. Exception: the `hero` shadow uses primary color at low opacity for CTA glow.

### File: constants/glass.ts
The "Liquid Glass" treatment for floating elements:
```
liquidGlass: {
  background: 'rgba(255, 255, 255, 0.7)',   // semi-transparent surface
  backdropBlur: 32,                           // heavy blur (use BlurView intensity)
  specularHighlight: {                        // 1px white top border — mandatory
    borderTopWidth: 1,
    borderTopColor: 'rgba(255, 255, 255, 0.4)'
  }
}

liquidGlassDark: {
  background: 'rgba(29, 27, 24, 0.85)',
  backdropBlur: 40,
  specularHighlight: {
    borderTopWidth: 1,
    borderTopColor: 'rgba(255, 255, 255, 0.15)'
  }
}
```
The specular highlight goes on the TOP edge ONLY. Adding it to all four sides flattens the glass effect.

### Navigation Bar: Floating Pill
The tab bar is NOT a standard iOS bottom bar. It is a FLOATING PILL positioned 32px from the bottom of the viewport.
- Shape: pill (border-radius: 9999)

- Active tab: terracotta tint background (primary at 10% opacity), primary text color, scale 1.1

## NON-NEGOTIABLE DESIGN RULES

1. NO solid borders between sections. Use background color shifts + whitespace.
2. NO pure black (#000000) anywhere. Text uses onSurface (#1D1B18).
3. ALL floating elements use Liquid Glass treatment (blur + specular highlight).
4. The tab bar is a FLOATING PILL, not a standard bottom bar.
5. The "Caption over Title" pattern (labelLarge above headline) is used for every section header.
6. Terracotta primary appears ONLY on interactive elements and accents.
7. Food photography is the hero — images large, cards generous, chrome minimal.
8. Noto Serif for headlines/display. Inter for body/labels/buttons. Never swap these roles.
9. All spacing on 8pt grid. No arbitrary pixel values.
10. Cards use rounded-xl (32px) or rounded-[28px]. Small elements use sm (8px). Never mix these scales.
11. Shadow color is tinted onSurface (#1D1B18), NEVER pure black.
12. Specular highlight (1px white top border) on TOP edge only of glass elements.
13. Respect Reduce Motion — all animations become instant crossfades.
14. Minimum touch target: 48pt on all interactive elements.
15. Dark mode must work. Dark surface is #161412 (warm near-black).
